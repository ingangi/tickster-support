TICKSTER

Pixel Dreams Reminder — macOS Menu Bar App

ABOUT

Tickster is a lightweight menu bar reminder app for macOS. It helps you stay on top of recurring tasks — hydration breaks, weekly meetings, bill payments, and more — with a retro pixel-art interface and real-time countdown widgets.

Task Types

One-TimeSet a single reminder
DailyRepeat every day
WeeklySpecific days of the week
Monthly / YearlySame date each cycle
Custom IntervalEvery N minutes
Exclusion RulesSkip specific dates or time ranges

More Features

  • Menu bar popover with live countdown to next task
  • Desktop widgets (Small, Medium, Large) with real-time progress
  • List view and grid (mosaic) view for task management
  • Notification actions: Postpone 30 min or Mark as Completed
  • Deep link support (tickster:// URL scheme)
  • iCloud sync across your Macs (Pro)
  • Available in English, Chinese, Japanese, Korean, Spanish, French, German

FREE vs PRO

Tickster is free to use with up to 3 tasks. A one-time Pro unlock ($1.99) gives you:

  • Unlimited tasks
  • iCloud sync via CloudKit (private database)

SETUP

Notifications

When you first launch Tickster, it will ask for permission to send notifications. This is required for reminders to fire. If you denied it, you can enable it later in System Settings → Notifications → Tickster.

Widgets

Right-click your desktop → Edit Widgets → search for "Tickster" to add countdown widgets.

iCloud Sync

Pro users can enable iCloud sync from the menu. Make sure you are signed into the same Apple ID on all your Macs. Sync uses CloudKit private database — your data is encrypted and only accessible by you.

TROUBLESHOOTING

  • Notifications not appearing: Check System Settings → Notifications → Tickster. Ensure "Allow Notifications" is on and Alert style is set to "Alerts".
  • Widget not updating: Remove and re-add the widget. macOS caches widgets aggressively — a restart may help.
  • iCloud sync not working: Verify you're signed into iCloud and have internet access. Try toggling sync off and on from the menu.
  • App doesn't appear in Dock: Tickster runs as a menu bar app. Look for the clock icon in your menu bar.
